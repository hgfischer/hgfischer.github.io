ImageMagick bindings for Go

Sat Aug 10, 2013, 100 words, read in about 1 mins.
computer-graphics golang programming

A few months ago I’ve spent some days developing my first cgo binding and the chosen library was ImageMagick’s MagickWand.

It was an interesting experience. It’s relatively very easy to do such thing in Go. I’m still not using this binding in production and I hope someone can try it. I took care to port some examples from C to Go also, so that should be easier to start using it. And since ImageMagick is a huge library I was not able to test every function through the binding.
